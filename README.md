# pony-cli-hello-world

## Description
This is a POC project to demonstrate pony.

## Tech stack
- ponyc
- pony

## Docker stack

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
